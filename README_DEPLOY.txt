Growing Shop live deploy bundle

Fastest route: GitHub Pages
1. Create a new GitHub repository.
2. Upload index.html and .nojekyll from this folder to the repo root.
3. In GitHub: Settings > Pages.
4. Under Build and deployment, choose "Deploy from a branch".
5. Select branch "main" and folder "/ (root)".
6. Save.
7. Your site will publish on your github.io URL.

Alternative:
Use a static host that accepts drag-and-drop deploys and upload index.html.

Files included:
- index.html
